# ScreenshotBot

Take Screenshot of your service and post your Slack channel.

## Getting Started

1.  Make a bot integration at [here](https://my.slack.com/services/new/bot).

2.  Add token into `config.json`

    ```json5
      {
        "token": "<slack token>",
        "imagePath": "/tmp", // optional
    }
    ```

3.  run `npm install`

4.  run `npm start`
